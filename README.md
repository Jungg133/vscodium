# My Twine Game

A text-based interactive story created with Twine.

## About

This is a private repository containing my Twine game. The game is an interactive fiction experience where players make choices that affect the story's outcome.

## How to Play

1. Open `index.html` (or your main game file) in a web browser
2. Read the story and click on highlighted text to make choices
3. Your decisions will shape the narrative and lead to different endings

## Files

- `index.html` - The main game file (exported from Twine)
- `assets/` - Contains images, audio, and other media files
- `README.md` - This file

## Development

This game was created using Twine and exported as HTML. To modify the game:
1. Import the HTML file back into Twine
2. Make your changes
3. Re-export as HTML
4. Commit and push the changes
